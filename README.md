# Django Langchain AI Webapp

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://chat.openai.com/c/LICENSE)

## Description

A Chatbot that answers questions about a given website.

## Installation

### Prerequisites

- Python 3.x
- Django
- Openai
- Tailwind CSS
- Tiktoken
- Langchain 
- Chromadb
